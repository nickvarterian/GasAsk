GasAsk
======
**GasAsk** is a web application developed to help improve the daily lives of people living in Los Angeles. GasAsk makes several calculations based on a userâ€™s wages, gas cost and vehicle mpg. GasAsk also calculates commute duration based on historic traffic conditions as well as the Uber cost for commutes. User's can compare different options for the most cost effective choice.
